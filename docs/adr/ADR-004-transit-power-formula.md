# ADR-004: Transit Power Formula

**Status:** Accepted
**Date:** 2026-04-15
**Constant:** AC-004

## Context
Card power must be derived from real astronomical data while remaining balanced for competitive play. The formula must be publicly verifiable — players can check the math.

## Decision
```
Transit Power = Base Floor (40) + Transit Bonus + Aspect Bonus + Dignity Bonus
Range: 40 (floor) → 100 (ceiling)
```
All figures share the same Base Floor (40). Power differences emerge entirely from real astronomical conditions.

## Consequences
- No inherent tier system — all figures are equal at baseline
- Power swings create strategic timing decisions
- The Visible Causality Display shows every component of the calculation
- Players can verify power against real planetary data
