# ADR-006: Forge Interaction Matrix

**Status:** Accepted
**Date:** 2026-04-15
**Constant:** AC-006

## Context
The five Forge schools need a balanced rock-paper-scissors-style interaction system that is easy to learn but creates meaningful counter-play.

## Decision
Each Forge is strong against one, weak against one, neutral against two (plus self-neutral). Modifiers are narrow (0.9×–1.1×) to prevent hard counters.

## Consequences
- No matchup is unwinnable — skill and timing dominate over pure matchup
- Players learn the matrix gradually through play
- The matrix is symmetric: if A is strong vs B, B is weak vs A
