# ADR-007: Celestial Energy Economy

**Status:** Accepted
**Date:** 2026-04-15
**Constant:** AC-007

## Context
The in-match resource economy must create meaningful decisions without being so tight that players feel locked out, or so loose that decisions are trivial.

## Decision
3 CE/turn base, max 1 carryover, +1 per active event (max 7). Bond activation is FREE — fate costs nothing. Chronos delay costs 2 CE — interfering with fate is expensive.

## Consequences
- 1 CE carryover (v3 addition) eliminates early-game lockout
- CE scarcity forces prioritization: summon vs attack vs delay
- Live celestial events create CE surges that change the tactical landscape
